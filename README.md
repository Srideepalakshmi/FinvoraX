# FinvoraX - Financial Dashboard

## 🚀 Overview
**FinvoraX** is a modern, high-performance financial intelligence dashboard architected for professional asset management. It focuses on clean data visualization, real-time analytics, and role-based security, providing users with a comprehensive view of their financial health.

Developed by: **Srideepalakshmi S**

---

## ✨ Features
- **Comprehensive Financial Tracking**: Real-time monitoring of **Total Balance**, **Income**, **Expenses**, and **Savings**.
- **Specialized Analytics**: Deep insights into spending patterns, highlighting peak expense categories like **Travel**.
- **Advanced Data Management**: 
  - Effortless **CRUD** operations with a refined modal interface.
  - Multi-column sorting and category-based filtering.
  - Global real-time search across all records.
- **Enterprise-Grade Security**: Integrated **Role-Based Access Control (RBAC)** (Admin vs. Viewer modes).
- **Intelligent Financial Scoring**: Real-time evaluation of financial stability and savings benchmarks.
- **Persistent Storage**: Data syncs automatically with `localStorage` for cross-session continuity.
- **Responsive Experience**: Optimized for flawless operation across Mobile, Tablet, and Desktop.
- **Dark Mode Optimization**: A tailored, accessible dark theme for optimized focus.
- **Data Export**: Professional CSV generation for offline auditing using the **Export Transactions** feature.

---

## 🛠️ Tech Stack
- **Framework**: [React 18](https://reactjs.org/)
- **Build Core**: [Vite](https://vitejs.dev/)
- **Styling UI**: [Tailwind CSS](https://tailwindcss.com/)
- **Iconography**: [Lucide React](https://lucide.dev/)
- **Visualization**: [Recharts](https://recharts.org/)
- **State Engine**: React Context API
- **Persistence**: Web Storage API (localStorage)

## 🏗️ How to Run
1. **Clone the repository:**
   ```bash
   git clone https://github.com/Srideepa123/FinvoraX.git
   cd FinvoraX
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Launch Development Server:**
   ```bash
   npm run dev
   ```
